# Expense Tracker (Java)
A simple Java console-based Expense Tracker application that helps users manage their daily expenses. The application supports adding, viewing, searching, updating, deleting expenses, and stores data permanently using file handling.
## Features
- Add Expense
- View Expenses
- Search Expense by ID
- Update Expense
- Delete Expense
- Calculate Total Spending
- Automatic Data Saving
- Automatic Data Loading
## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- ArrayList
- File Handling
- Serialization
- VS Code
- Git & GitHub
## How to Run
Compile:
javac Main.java Expense.java
Run:
java Main
